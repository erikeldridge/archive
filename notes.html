An Introduction to Web Development

HTML: the foundation of web pages

  1. Open foundation.html
    • Foundation.html contains only HTML, the basic structure for information on a web page
  2. Right click on the foundation.html page and select View Page Source, to see the HTML
    • Notice that the code tells a story about the information on the page.
    • The <html> tag says "this is html".
    • The <body> tag says "this is the body of this document".
    • <div> tags define a division, or section, in the document.
    • The text class="image" within a <div> tag describes the division as containing an image
  3. Close the window containing the foundation source
  4. Browse to Facebook and view the page source
    • Even though the code is very complicated, you can see commonalities with the basic code: <html>, <body>, etc
  5. Close the window containing the Facebook source

Styling: making a web page look good

  1. Go back to foundation.html and click the Styling: making a web page look good link at the bottom to go to styling.html
    • Notice that the way the information is being presented has changed.
    • Colors have been added.
  2. View the source of this page.
    • There's a new <style> tag. Within the <style> tag there are rules describing how the page should look.
    • The code inside the <style> tag is called CSS
    • Rules affecting tags are named with the name of the tag, so the rule starting with body { affects the <body tag.
    • The lines beginning with /* are called comments. They provide a way for the person who built the page to explain what she was doing
  3. Open a new window, browse to Wikipedia's English site, view the page source, look for a tag named link, and click on the link in it, eg http://cotnet.diggstatic.com/css/368/global.css
    • Find some similarities between the simple styles in styling.html and the styles for Digg.
    • Do you see body { and /*?
  4. Switch back to styling.html
  5. In the toolbar, go to Tools > Firebug > Open Firebug (if you don't have Firebug, you can get it from the Firebug site)
  6. In the panel that opens at the bottom of the page, click the inspect button next to the "firebug"
  7. Move the mouse pointer over an image on the page
    • You should see a blue outline surround the image (or element) your pointer is over
    • In the firebug panel, you should see an <img tag highlighted in turquoise
    • On the right-hand side of the panel, you should see the style rule being applied to the <img> tag. Hint: it starts with img {
  8. Click the mouse button on the image to fix the inspection on a particular image
  9. In the style rule for image, click the text 10px solid black next to the word "border"
    • It should change into something you can type into
  10. Change the word "black" to "red"
    • Note that the color of the border around all the images is now red
  11. Play around with the other styles.
    • Refresh the page to reset the styling
  12. Switch back to Digg and play with the styling there
    • You may need to click the Apply settings for digg.com button in Firebug
  13. On your computer's desktop there is an icon for Notepad. Double click it to launch Notepad
  14. In Notepad, open styling.html in a folder called mbits on your desktop
  15. Find the definition for the img style rule and change the border color to "green"
  16. In Firefox, refresh styling.html and observe the borders are now all green
  17. Refresh the page and observe the changes persist
    • You are now web programmers
    • If you put this page on a server, you would have

Scripting: making a web page interactive

  1. In Firefox, click the Scripting: making a web page interactive link at the bottom of styling.html
    • Observe that there is now only one image displayed
  2. Click the arrows under the picture to rotate through the images
  3. View the source and scroll to the bottom of the code
    • Observe there is now a <script> tag in the HTML
    • The code inside the <script> tag is called JavaScript
    • Nothing else in the file has changed, ie the HTML and CSS are the same, and JavaScript has changed the way we interact with the content
    • step through the script
      1. get all div tags on the page
      2. extract only the ones that have the class image
      3. define functions to show and hide images. Functions provide a way of doing something repeatedly w/o having to copy code
      4. define a function to respond to mouse clicks on the arrow links
      5. hide all the images
      6. create the arrow links and configure them to point to the right place
      7. show the first image
  4. Place the mouse pointer over one of the arrows, right click, and select inspect element
  5. Now click on the arrow and observe the changes to the code in the Firebug menu
  6. Open a window for Flickr
  7. Search for a picture of a something, eg a panda, and click one of the pictures in the results
    • On the right, you'll see two small images with left and right arrows
  8. Right click on one of the image and inspect it
  9. Click on the arrows and observe the changes in the code
    • This is a much more complex piece of code, but it shares the common principle of using JavaScript to make changes to the information being displayed
  10. As a last exercise, click the "Console" button in Firebug
  11. At the bottom of the panel, click to the right of the 3 greater-than symbols, type document.getElementsByTagName('div');, and press enter
    • Observe the results in the Firebug panel above where you typed
    • It will output about a million results
  12. Hit the up arrow and edit your code to be document.getElementsByTagName('div')[0];
    • This will select just the first div tag on the page
  13. Hit the up arrow and edit your code to be document.getElementsByTagName('h1')[0];
    • This will select just the first h1 (header) tag on the page
  14. Place your mouse pointer over the result
    • Observe the title of the image on the page be highlighted in blue and yellow
  15. Click back in the code input box, hit the up arrow again, and edit your code to be document.getElementsByTagName('h1')[0].innerHTML = 'panda';
    • Observe the title change on the page
    • Can you imagine how you might build a way to allow someone to change the title of their picture?
